Hey, I am working on WineMorpher, a WoW 3.3.5a morphing tool built for Wine and WoWSilicon on macOS.

## More Technical Version

I build experimental tools for older World of Warcraft clients, with a focus on WoW 3.3.5a running through Wine and WoWSilicon on macOS.

My current project is WineMorpher: a local visual appearance morpher made from a WoW addon UI plus a native Wine-loaded DLL. It aims to bring a Transmorpher-style experience to macOS players, including item previews, gear morphs, mounts, pets, race/NPC morphs, titles, weapon enchants, sets, and loadouts.

Supporting this page helps fund the time spent reverse engineering, testing, packaging releases, writing docs, and making the project easier for non-developers to install and use.
